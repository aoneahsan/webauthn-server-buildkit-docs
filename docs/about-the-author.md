## The project

webauthn-server-buildkit is a framework-independent WebAuthn / FIDO2 / passkeys server library for TypeScript and Node.js. It pairs naturally with the companion Capacitor plugin [`capacitor-biometric-authentication`](https://www.npmjs.com/package/capacitor-biometric-authentication) but works in any Node.js backend.

- **Source code:** [github.com/aoneahsan/webauthn-server-buildkit](https://github.com/aoneahsan/webauthn-server-buildkit)
- **npm package:** [npmjs.com/package/webauthn-server-buildkit](https://www.npmjs.com/package/webauthn-server-buildkit)
